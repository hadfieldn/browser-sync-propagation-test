BrowserSync Event-Propagation Demo

This demo shows that when pages are served by BrowserSync, events are propagated among all browsers that are opened to the same URL.

Instructions

  1. Open another browser window to this same URL.
  2. Click the Send Signal/Clear Signal buttons on any of the windows that are open to this page. If the page is being served through BrowserSync (gulp serve), all the other windows will receive the signal.
  3. In the terminal window from which BrowserSync is serving this page, terminate BrowserSync (Ctrl-C), and enter http-server -p <port> to serve this page again without BrowserSync.
  4. Now click the Send Signal button again. This time, the signal will NOT propagate to the other windows.
